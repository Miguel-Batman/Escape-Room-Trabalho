# Escape da israely
## História
A professora israely passou um trabalho de informatica basica, mas voce(jogador) ficou lesando e não fez o trabalho. O tempo limite pra entregar acabou, e agora vc ta marcado pela israely. Se voce quiser continuar com vida, deve buscar uma forma de se livrar da israely.
### Oque fazer?
Pra sair com vida, voce deve buscar uma forma de montar algo pra levar a israely pra longe de vc, de uma forma que ela nunca mais volte pra te atormentar(ou quem sabe ela volte na parte 2!?)
